Installation
Clone the repository
bash
git clone https://github.com/SanchitKumaRR/scal.git
cd scal
Make the script executable
bash
chmod +x scal.sh
Run directly
bash
./scal.sh
Optional: Install as a global command
Create a symbolic link so you can use scal from anywhere:

bash
sudo ln -s /full/path/to/scal/scal.sh /usr/local/bin/scal
Replace /full/path/to/scal/ with the actual path to your cloned repository.

Then run the command by typing:

bash
scal
Usage
Follow the on-screen prompts to perform calculations.

Packaging with Pacman
This repository includes a PKGBUILD file for Arch Linux users.

Build the package with:

bash
makepkg
Install the package with:

bash
sudo pacman -U scal-1.0-1-any.pkg.tar.zst
Then run the command:

bash
scal
Dependencies:
bash (usually pre-installed)
bc
cmatrix:
Arch based - sudo pacman -S cmatrix
Debian based - sudo apt install cmatrix
